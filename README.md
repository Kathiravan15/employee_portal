# Employee Portal (Full-Stack Assignment)

A full-stack employee management portal built using:

- **Frontend:** React + Vite + TailwindCSS + Redux Toolkit  
- **Backend:** Node.js + Express + MongoDB  
- **Auth:** JWT, bcrypt  
- **RBAC:** Admin & Supervisor roles  
- **Database:** MongoDB (local or Atlas)

---

## Features

### Auth
- JWT-based login
- Role-based access control
- Admin can create supervisors
- Supervisors can log in and manage employees they created

### Employees
- Create, Read, Update, Delete (CRUD)
- Pagination & search
- Supervisor can view **all employees**
- Supervisor can update only their employees
- Delete restricted to admin
